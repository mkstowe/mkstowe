Seasonal Emotional Disorder Detection

A machine learning-oriented Python program that scans and analyzes Twitter users' tweets to find trends in sentiment to predict the possibility of seasonal emotional disorders such as Seasonal Affective Disorder (SAD) or Bipolar Disorder.

Spoken Word Calculator

A Python program that uses Hidden Markov Models to determine the words spoken in a sound clip, allowing a user to perform calculations using only their voice.

Network Security

A collection of programs written in Go to both protect against and exploit vulnerabilities in HTTP and DNS servers. The protection aspect involved detecting spoofed ARP packets and the use of port scanning, and the malicious aspect involved spoofing DNS packets to expose man-in-the-middle vulnerabilities.

Web Security

A collection of web-based programs to expose common vulnerabilities including SQL injection, XSS using HTML and JavaScript, and CSRF.

Crypto

A collection of cryptography programs written in Python including length extension attacks, hash collisions, RSA signature forgery, padding oracle attacks, and hacking the Vigenère cipher.

Schedule Recommender

A recommender system written in Python that uses machine learning to take in the classes a student as taken so far, find previous students who have taken similar classes, and output a recommended schedule consisting of classes the student would probably enjoy.

LC2K

A simulation of the Little Computer 2000 (LC2K) written in C. This simulated computer included an assembler to convert assembly code for a custom ISA to machine code, single cycle, multi cycle, and pipelined processors, a linker, and a cache system.

Donut Shops

A C++ program that utilizes Prim's Algorithm and heuristics to efficiently find approximate solutions to the Travelling Salesperson problem (with donut shops!)

theCurlyBot

A reddit bot - called theCurlyBot - that I created with a small team using Python. Our bot pulls fake news headlines from the subreddit /r/TheOnion, and real (but fake-sounding) headlines from /r/NotTheOnion. It then runs a natural language processing algorithm on the pool of headlines that it continually collects, and makes its own fake news headlines that it then posts to its own subreddit.

Image Resizing

A program written in C++ that resizes images without cropping them. It inputs the original image, and removes rows and columns of unneeded pixels based on the pixels that surround it until reaching the desired size. This allows the image to be resized without much noticeable change in terms of content. It also allows for images of faces to be hilariously distorted by resizing beyond a reasonable amount.

Puzzle Solver

A C++ program that uses breadth-first search and depth-first search algorithms to solve three-dimensional mazes. Starting on the space labelled "@", the program finds a route to the space labelled "?". In order to go through doors (A, B, C, etc.), it must first go to the associated buttons (a, b, c, etc). However, if it encounters a "^" space, all of the doors close.

Galaxy Battles

A C++ program that takes in various Jedi and Sith battalions with different force and troop values, then uses heaps to play out their battles.

Log Manager

A C++ program that uses hash tables to allow a user to create and edit log entries.

Mechanical Keyboard

I designed, gathered parts for, and built a custom mechanical keyboard. The electronics are completely hand-wired to an Arduino Teensy. I coded the firmware to match the custom key layout that I use. The parts for the casing were laser cut to fit all of the inner components.
